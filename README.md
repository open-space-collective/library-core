## Structure

The **Core** library exhibits the following structure:

```txt
├── Types
│   ├── Integer
│   ├── Real
│   ├── String
│   ├── Byte
│   ├── Byte Array
│   ├── Index

│   └── Weak
├── Containers
│   ├── Array

│   ├── Object
│   ├── Dictionary
│   ├── Pair
│   ├── Triple
│   ├── Tuple

│   └── Tree
├── File System
│   ├── Path
```
